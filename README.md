# Pokedex

## Install

To install clone this repo from my github:
```shell
gh repo clone msquinn/bitly-interview
```
or 
```shell
git clone https://github.com/msquinn/bitly-interview.git
```

To run the pokedex first install: 

```shell
npm i
```

Then run:

```shell
npm run dev
```


## Tests

To run tests:

```shell
npm run test
```

## Production

To run a production build:

```shell
npm run build
```

To view:

```shell
npm run start
```
